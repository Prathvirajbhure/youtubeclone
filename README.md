 YouTube Clone UI 

📺 YouTube Homepage Clone (HTML & CSS)

A YouTube homepage UI clone built using HTML and CSS.
This project recreates the basic layout of the YouTube homepage including the header, sidebar, video grid, and thumbnails.

It is a frontend practice project focused on learning layout design, flexbox, grid, and responsive UI development.

⸻

✨ Features
	•	📺 YouTube-style homepage layout
	•	📂 Sidebar navigation menu
	•	🔍 Search bar with search button
	•	🎤 Voice search icon
	•	🔔 Notification icon with counter
	•	🖼️ Video thumbnail grid
	•	📱 Responsive design
	•	👤 Channel profile icons
	•	⏱️ Video duration overlay

⸻
